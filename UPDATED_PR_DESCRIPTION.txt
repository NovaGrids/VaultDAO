## ğŸ”” Notification Center System

A fully functional notification center that replaces basic toast notifications with a persistent, categorized system.

**Key Features:**
- **Categorization**: Proposals, approvals, and system notifications
- **Priority Levels**: Critical, high, normal, and low with visual hierarchy
- **State Tracking**: Read/unread status with visual indicators
- **Actions**: Inline approve, view, and dismiss actions with async support
- **Persistence**: LocalStorage with 500 notification limit
- **History**: Full notification history with pagination (20 per page)
- **Filtering**: By category, priority, and read status
- **Sorting**: By timestamp or priority (ascending/descending)
- **Mobile**: Swipe-to-dismiss gestures, responsive design
- **Accessibility**: Full ARIA support, keyboard navigation, screen reader friendly

**Components:**
- `NotificationCenter.tsx` - Main notification panel
- `NotificationItem.tsx` - Individual notification display

- `NotificationContext.tsx` - Global state management
- `useNotificationCenter.ts` - Convenience hook

## ğŸ“Š Proposal Comparison Tool

A sophisticated comparison system for analyzing multiple proposals side-by-side with duplicate detection.

**Key Features:**
- **Multi-Select**: Compare up to 5 proposals simultaneously
- **Side-by-Side View**: Horizontal scrollable comparison table
- **Diff Highlighting**: Visual differences with insert/delete/equal markers
- **Similarity Detection**: Levenshtein distance algorithm
- **Duplicate Detection**: Automatic alerts for >85% similarity
- **Similarity Matrix**: Visual similarity scores between all proposals
- **PDF Export**: Professional formatted PDF with jsPDF
- **Mobile**: Responsive with sticky headers and horizontal scrolling
- **Performance**: Optimized with useMemo for calculations

**Components:**
- `ProposalComparison.tsx` - Selection interface
- `ComparisonView.tsx` - Side-by-side display
- `SimilarityDetector.tsx` - Duplicate detection and alerts

**Utilities:**
- `similarityDetection.ts` - Levenshtein algorithm implementation
- `diffHighlighting.ts` - Diff calculation and segment merging
- `pdfExport.ts` - PDF generation with proper formatting
